# Sample Images

Place test images in this directory for testing the 3D model generation.

Recommended image characteristics:
- Single objects (like a chair, car, or toy)
- Clear background if possible
- Good lighting
- Reasonable resolution (at least 300x300 pixels)
- Common formats: JPG, PNG

Example images that work well:
- A simple toy car against a plain background
- A chair photographed from the front
- A coffee mug on a table
- Simple geometric objects
